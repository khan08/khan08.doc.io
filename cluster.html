cluster.ec2 module¶

cluster.ec2.launch(instance_type, keyname, key_filename, workers=0, ami_id='ami-f4cc1de2', sgs=['default'], browser=True, mode='single')[source]¶

This function helps user to launch a dask cluster on aws. It will launch a t2.micro type instance as the manager node, and launch custom instances as worker nodes.

Parameters:
  • instance_type – The type of the worker instances
  • keyname – The ssh key name of the manager instance
  • key_filename – The .pem path. For example, /.ssh/mykey.pem
  • workers – The number of worker instances in dask cluster.For now, it is ignored when mode is not dask
  • ami_id – The operating system of the worker instances. Defaults to ubuntu, if mode gpu then ignore and use bitfusion image
  • sgs – The security group of the manager and worker instance
  • browser – Defaults to True. Open notebook in browser, no browser if set to False
  • mode – Options: ‘dask’,’single’. Defaults to single.
Returns:

returns manager instance with definition as boto3.EC2.instance
