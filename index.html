Welcome to daskEc2’s documentation!¶

Introduction¶

Create a dask cluster using docker on AWS. You need to use cluster.ec2.launch().

Quick Start¶

Setup the einvironment. Install boto3 and setup aws access key and secret

pip install boto3

Go to ~/.aws/credentials and set the key and secret

[default]
aws_access_key_id='YOUR_KEY'
aws_secret_access_key='YOUR_SECRET'

Checkout the code for cluster launching from private repo.

svn checkout https://weih.unfuddle.com/svn/weih_cheq/branches/daskec2/clusterAPI/cluster [local-path]

Run the launcher

from cluster import ec2
manager = ec2.launch(n=1, instance_type='t2.micro')

Browse to notebook when manager is ready. https://manager.public_ip_address:8888
