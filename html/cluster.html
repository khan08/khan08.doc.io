cluster.ec2 module¶

cluster.ec2.launch(n, instance_type, image_id='ami-f4cc1de2', keyname='dask', sgs=['default'])[source]¶

This function helps user to launch a dask cluster on aws. It will launch a t2.micro type instance as the manager node, and launch custom instances as worker nodes.

Parameters:
  • n – The number of worker instances
  • instance_type – The type of the worker instances
  • image_id – The operating system of the worker instances
  • keyname – The ssh key name of the manager instance
  • sgs – The security group of the manager and worker instance
Returns:

returns manager instance
